Data Analyzer

This project is a simple data analysis tool that loads a dataset, cleans it, analyzes it, creates visualizations, and saves a cleaned version of the data. It is designed to work in Google Colab or a local Python environment.

Features

Load CSV or JSON files

Clean missing numeric and categorical values

Remove duplicate rows

Analyze numeric and categorical columns

Generate histograms, boxplots, and a correlation heatmap

Print summary statistics and most common categories

Save the cleaned dataset to a new file

What the Analyzer Does

Uploads or loads your dataset

Cleans the data by handling missing values and duplicates

Analyzes numeric and categorical columns

Displays visualizations to help understand the data

Prints a summary of important findings

Saves the cleaned data as a new file

Output You Receive

First five rows of the dataset

Summary statistics for numeric columns

Value counts for categorical columns

Histograms of numeric columns

Boxplots comparing numeric distributions

A correlation heatmap

A file named cleaned_data.csv

Requirements

This tool uses the following Python libraries:

pandas

numpy

matplotlib

seaborn

google.colab (only when uploading files in Colab)

How to Use

Run the script

Upload a CSV or JSON file when prompted

Wait for the analysis to complete

Download the cleaned data file
